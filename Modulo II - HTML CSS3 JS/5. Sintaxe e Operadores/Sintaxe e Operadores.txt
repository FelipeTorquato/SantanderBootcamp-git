1.
    -> Objetivos:
        Introduzir os principais operadores para manipular valores no seu código.
    -> Tipos de operadores:
        • Aritmética
        • Atribuição
        • Comparação
        • Lógica
        • Condicional
    -> Atribuição:
        • =     x = y   x = y
        • +=    x += y  x = x + y
        • *=    x *= y  x = x * y
        • /=    x /= y  x = x / y
        • %=    x %= y  x = x % y
    -> Aritmética:
        • +     Adição
        • -     Subtração
        • *     Multiplicação
        • **    Exponencial
        • /     Divisão
        • %     Módulo
        • ++    Incrementar
        • --    Decrementar
    -> Comparação:
        • ==    Igual a
        • ===   Mesmo valor e mesmo tipo
        • !=    Diferente
        • !==   Valor e tipos diferentes
        • >     Maior que
        • <     Menor que
        • >=    Maior ou igual
        • <=    Menor ou igual
    -> Lógica:
        • &&    "e" lógico
        • ||    "ou" lógico
        • !     "não" lógico
    -> Condicional:
        • Ternário: (condition) ? expression1 : expression2
