2.
    -> Javascript é uma linguagem de tipagem dinâmica e fracamente tipada. Ou seja, antes de declarar um valor, você não especifíca o tipo dele.
    -> Tipos de dados
        -> Primitivos:
            • numbers.
            • strings.
            • boolean.
            • null.
            • undefined.
        -> Não-Primitivos:
            • objects.
            • arrays.
    -> Strings
        • Comumente utilizadas para textos.
        • Valores declarados entre aspas ou crases.
    -> Numbers
        • Números inteiros ou decimais.
        • Biblioteca Math.
    -> Boolens
        • True or false.
    -> Arrays
        • Lista iterável de elementos.
    -> Objetos
        • Estrutura do tipo "chave e valor".
